# The Logic Forge - Problem Set Solutions

## About
This repository contains my solutions for **The Logic Forge**, a 3-week logic and problem-solving challenge organized by the CIS Community, NED University of Engineering and Technology.

## Activity Details
- **Duration:** 3 weeks
- **Start Date:** December 29, 2024
- **Organized by:** CIS Community, NEDUET
- **Objective:** Enhance logical thinking and problem-solving skills through weekly challenges
